\subsubsection{Flexi Swap}
\label{pricing:ir_flexiswap}

Flexi Swaps are priced following the Replication Approach described in (F. Jamshidian, 2005). The replicating basket of
Bermudan Swaptions is priced using the methods as described under Product Type ``Bermudan Swaption'', directly routing
the components of the basket to pricing engine config for product type ``BermudanSwaption''
resp. ``BermudanSwaption\_XCcy''.
