# ORE Documentation

- [User Guide](userguide.pdf): Start here - introduction, installation instructions, example use cases, parametrisation

- [Products](products.pdf): Full catalogue of supported financial instruments and pricing models, including scripted payoffs

- [Methods](methods.pdf): Summary of simulation, xva and risk methods used in ORE

- [Design](ore_design.pdf): Introduction into the internal workings of ORE's stack of libraries

- [American Monte Carlo](amc.pdf): Implementation of American Monte Carlo simulation in ORE

- [Scripted Trade](scriptedtrade.pdf): Scripted trade framework, representation of Exotics using ORE's payoff script language

- [Credit Model](creditmodel.pdf): A proof of concept credit portfolio model implementation in ORE inspired by Credit Metrics, integrating credit and market risk

- [Formula Coupon](formulabasedcoupon.pdf): Predecessor of the scripted trade framework, supporting composite coupons with multiple indices
